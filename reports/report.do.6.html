Analysis specification

Analysis was done on a Digital Ocean VPS instance in Amsterdam Netherlands with following specifications:

Proxy Hardware:
1 x86 Core
512MB memory
Testing config:
6 workers
spawning 200 phones

Response Times plot shows the response times for individual transactions (REGISTER, INVITE, BYE) befined as the time from sending a request to the first meaningfull response:

Request Meaningfull Response
Unauthorized REGISTER 401
Authorized REGISTER 200
INVITE 200
BYE 200

X axis for Histogram and Rasponse Times plots are in milliseconds.

Rasponse Times plot shows each separate dataset as a bar with a Minimum value on the left side, Maximum value on the right side, Mean value delineated by the black line and with red lines depicting the Population Standard Deviation.

Histograms depict the distribution of transaction response times while Transaction plots show the duration of each Transaction state as well as transitions between different states.

The density of transition lines in Transaction plots shows the distribution of Transaction state durations which is allowed by the transparency setting of said lines.

Please note that all Calls should last a minimum of 10 seconds which is the value for the Auto hangup timer. Everything above this value are delays caused by the load on the system.
